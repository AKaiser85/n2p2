nnp-tool
========

Usage:
------

.. code-block:: none

   nnp-tool <arg1> <arg2>
              <arg1> ... Argument 1 meaning.
              <arg1> ... Argument 1 meaning.
              Execute in directory with these NNP files present:
              - input.data (structure file)

Examples:
---------

  .. code-block:: none

     nnp-tool random 0.1 123

Sample screen output:
---------------------

.. code-block:: none

   *** NNP-TOOL ******************************************************************

   *******************************************************************************

File output:
------------

* ``output.data``\ : The requested subset of training structures.
